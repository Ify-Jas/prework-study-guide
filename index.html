HTML

  • HTML primarily is responsible for the structure of the webpage
  • HTML stands for Hypertext markup language.
  • The head element contains information about the webpage.
  • The body element represent the visible content shown to the user
  • We use the special tags for events such as paragraphs(to give space) to start a line and break to break into another line
  • We also have an opportunity to add red links to hyperlink and images by correctly linking the source

CSS

  • CSS is a programming language used to style or beautify a wepbage.
  • CSS stands for cascading style sheets
  • There are 3 ways of using CSS to style a webpage
    1. Inline
    2. Internal
    3. External
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS codes are generally linked to the "Head" of the index.html

Git

  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • JavaScript is a high-level programming language that allows user to interact with the website.
  • JavaScript takes a static HTML/CSS page and makes it interactive.
  • JavaScript codes are generally linked to the "body" of the index.html.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are use in JavaScript to improve reusability and reduce repetition, it saves time too.
  • To call a function, functionname();
